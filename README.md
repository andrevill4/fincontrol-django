#  FinControl - Sistema de Controle Financeiro

Sistema de controle financeiro desenvolvido com Django, permitindo o gerenciamento de receitas e despesas com dashboard interativo.

##  Funcionalidades

- Cadastro de receitas
- Cadastro de despesas
- Dashboard com:
  - Total de receitas
  - Total de despesas
  - Saldo atual
- Filtro por tipo (Receitas / Despesas)
- Interface organizada e intuitiva

##  Tecnologias Utilizadas

- Python
- Django
- SQLite
- Bootstrap

##  Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/andrevill4/fincontrol-django.git

# Entre na pasta
cd fincontrol-django

# Crie um ambiente virtual
python -m venv venv

# Ative o ambiente
venv\Scripts\activate

# Instale as dependências
pip install -r requirements.txt

# Execute o servidor
python manage.py runserver
```
